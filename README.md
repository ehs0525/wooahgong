## Commit Convention
1. 제목에서 커밋 종류는 영어로 그 외는 한글로 작성
2. 제목과 본문은 한 줄 띄워 분리
3. 제목 끝에 . 금지
4. 본문은 어떻게보다 **무엇**을, **왜**에 맞춰 작성하기

#### Convention
```
Add      -  코드 추가
Update   -  코드 수정
Remove   -  코드 삭제
Fix      -  버그 수정
Rename   -  단순 이름 변경
Docs     -  문서 관련
```

#### Example
```
[Add] 로그인 유효성 검사 추가
[Docs] API 명세서 수정
```
